Вбудовані елементи HTML

HTML визначає довгий список доступних вбудованих тегів, повний список яких можна знайти на Mozilla Developer Network.

  • Для виділення тексту жирним використовуйте <strong>.
  • Для виділення тексту курсивом використовуйте <em>.
  • Скорочення, як-от HTML, повинні використовувати <abbr>, з необов'язковим атрибутом title для повної фрази.
  • Цитати, як-от — Марк Отто, повинні використовувати <cite>.
  • Видалений текст слід позначати через <del>, а доданий текст — через <ins>.
  • Текст у верхньому індексі текст використовує <sup>, а текст у нижньому індексі текст використовує <sub>.

Більшість з цих елементів стилізуються браузерами з мінімальними змінами з нашого боку.
